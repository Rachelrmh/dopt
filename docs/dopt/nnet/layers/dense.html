    Function dense

    Creates a fully connected layer.

    Layer dense (
      const(Layer) inputs,
      ulong units,
      void delegate(dopt.core.ops.Operation) weightInit = glorotGaussianInit(),
      void delegate(dopt.core.ops.Operation) biasInit = constantInit(0.00000F)
    );

    If biasInit is set to null, then no biases are added to the output.

    Parameters

    NameDescription
    inputs The input to the new layer.
    units The number of fully connected units in the new layer.
    weightInit A ParamInitializer used to initialise the weight matrix.
    biasInit A ParamInitializer used to initialize the bias vector.

    Returns

    A new Layer representing a linear transform.
