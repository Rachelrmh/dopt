    Function slice

    Slices the result of an operation.

    dopt.core.ops.Operation slice (
      const(dopt.core.ops.Operation) input,
      const(ulong)[] start,
      const(ulong)[] stop,
      string mod = __MODULE__,
      ulong line = cast(ulong)__LINE__
    );

    Parameters

    NameDescription
    input The operation that should be sliced.
    start The starting indices for each dimension.
    stop The stopping indices for each dimension.

    Returns

    The new Operation.

    Example

    import dopt.core.cpu : evaluate;

auto s1 = int32([3, 3], [
    1, 2, 3,
    4, 5, 6,
    7, 8, 9
]).slice([1, 1], [3, 3]);

assert(s1.evaluate().as!int == [
    5, 6,
    8, 9
]);
